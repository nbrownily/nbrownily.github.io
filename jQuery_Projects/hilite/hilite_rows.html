Web Development Certificate
The web development curriculum at Aims is comprised of 4 classes covering the key areas of web site development.The student will be exposed to the entire gamut of web development topics from basic HTML layout methods, tags, and cascading style sheets to creating dynamic web pages using Javascript and server-side scripting for database interaction using PHP and MySQL.
Course ID Name Description Credits Prerequisites Syllabus
CIS 118 Intro. to PC Applications Introduces students to word processing, spreadsheets, database management, and presentation software. 3 Can be taken in conjunction with CSC119 and CIS124 Syllabus
CIS 124 Intro to Operating Systems Introduces the student to Windows and command line topics. 3 Can be taken in conjunction with CSC119 and CIS118 Syllabus
CSC 119 Logic and Program Design This is the entry level programming class. The student is introduced to computer program design using concepts of structured programming and logic. 3 Can be taken in conjunction with CIS124 and CIS118 Syllabus
CWB 110 Complete Web Authoring Explores the complete set of web development and layout methods using HTML and Cascading Style Sheets(CSS). 3 CIS118 or Instructor Permission Syllabus
CWB 204 Cascading Style Sheets This class explores the use of the Cascading Style Sheets (CSS) language as a means of separating content and design on web documents. 3 CWB 110 Syllabus
CWB 205 Complete Web Scripting Explores the use of client side scripting languages, HTML, and CSS to create web pages that will dynamically change in response to user actions. 3 CWB110,CSC119 Syllabus
CWB 206 Web Database The course uses the PHP scripting language, MySQL database server, and Apache Webserver. 3 CWB110, CSC119,CIS145 Syllabus
